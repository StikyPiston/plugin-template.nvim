# plugin-template.nvim

This is a simple template to get started with making a Neovim plugin.

Make sure to change the **LICENSE.md** file to the license you would like to
use (if you stick with the MIT license, remember to change the `Copyright` line
to your name)

Also, change the name of the directory inside the `lua` directory from `example` to the name
of your project.

## To use this template:

Simply select the **Use this Template** button at the top-right of the page,
and follow the steps to create a repository based on this template.
